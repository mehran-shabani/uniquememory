# بستهٔ تحویل نسخه GA

این دایرکتوری کلیه اقلام قابل تحویل برای انتشار General Availability را در خود جای داده است.


## فهرست اقلام
- `openapi/uniquememory.yaml`: مشخصات کامل API عمومی برای شرکای B2B.
- `mcp/manifest.json`: مانیفست MCP برای اتصال پلتفرم‌های حافظه مشارکتی.
- `sdk/python/` و `sdk/javascript/`: کیت‌های توسعه‌دهنده با مثال‌های شروع سریع.
- `security/security-whitepaper.md`: سفیدنامهٔ امنیت و انطباق.
- `GA_RELEASE_NOTES.md`: راهنمای تگ و انتشار نسخه GA.

هر تغییر در اقلام فوق باید با افزایش نسخه semantic و ثبت در مخزن انجام شود.

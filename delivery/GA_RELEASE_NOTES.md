# انتشار GA — نسخه 1.0.0

## مراحل انتشار
1. اطمینان از به‌روزرسانی همه اقلام در `delivery/` و اجرای تست‌های نهایی.
2. افزایش نسخه در بسته‌های SDK و فایل‌های OpenAPI/Manifest در صورت نیاز.
3. اجرای دستور زیر برای ساخت تگ:

   ```bash
   git tag -a v1.0.0 -m "UniqueMemory GA"
   git push origin v1.0.0
   ```
4. بارگذاری بسته‌ها در مخازن مربوطه (PyPI، npm) و اشتراک‌گذاری سفیدنامه امنیتی با شرکا.
5. ارسال اعلان GA برای مشتریان پایلوت و آغاز برنامه مقیاس‌پذیری.

## تغییرات کلیدی
- تثبیت APIهای جست‌وجو و ingest با بودجه Latency مستند.
- ارائه‌ی SDKهای رسمی Python و JavaScript.
- تکمیل مدارک امنیت و MCP Manifest برای اکوسیستم شرکا.
